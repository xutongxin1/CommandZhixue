This is a small programme to query score in Zhixuewang.

这是一个用来查询智学网上成绩的小脚本
现在由xutongxin完成图形化工程
Base on [Zhixuewang](https://github.com/anwenhu/zhixuewang), a API of Zhixuewang.

To run:
```bash
pip install zhixuewang
python Main.py
```

基于[某智学网API](https://github.com/anwenhu/zhixuewang)
